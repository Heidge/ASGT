 
# ASGT
 
ASGT is a Django application for managing tournaments.
 
 
## Tech Stack
 
**Client:** HTML, CSS, JavaScript
 
**Server:** Django, Flask
 
 
## Run Locally
 
Clone the project
 
```bash
  git clone https://link-to-project
```
 
Go to the project directory
 
```bash
  cd my-project
```
 
Install dependencies
 
```bash
  npm install
```
 
Start the server
 
```bash
  npm run start
```
